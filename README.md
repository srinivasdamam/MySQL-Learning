# MySQL Learnings

### Create Database
---

``` 
CREATE DATABSE <name>;
```

E.g

```
CREATE DATABSE pet_store;
```

 Note: Enclose the database name in quotes if name demands space character

```
CREATE DATABSE `Pet Store`;
```



### Drop Database
---
```
DROP DATABASE <name>;
```


### Use Database
---
```
USE <database name>;
```
